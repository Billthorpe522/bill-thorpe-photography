# Bill Thorpe Photography

Portfolio website for Bill Thorpe Photography — wedding, portrait, sports, and event photography serving Manhattan, IL and the Chicago area.

Built as a single-page HTML site with Shopify CDN-hosted images. Deployed via GitHub Pages.

## Sections
- Hero with stats (15+ years, 50+ events)
- Services & pricing (7 service cards)
- About / bio
- Filterable masonry gallery (24 images, 6 categories)
- Shop CTA linking to Shopify store
- Contact info

## Tech
- Single HTML file, all CSS/JS inlined
- Shopify CDN for image hosting
- CSS columns masonry layout
- Lightbox viewer with keyboard/touch support
